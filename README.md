# 网页自动刷新工具
一个简单易用的本地网页批量访问刷新工具，通过真实Chrome浏览器模拟用户操作，完全本地运行无任何数据上报。
## ✨ 功能特性
### 核心功能
- 🖥️ 友好的Web操作界面，无需命令行操作
- 🌐 自定义目标网址，自动补全http/https协议，支持短链接
- 🔢 可设置访问/刷新总次数
- ⏸️ 支持任务中途停止，立刻关闭所有浏览器
- 📊 实时进度显示，展示已完成次数/总次数/进度百分比
### 两种功能模式
1. **单窗口重复刷新（PV模式）**：启动一个浏览器窗口，重复刷新指定次数，仅增加页面访问量PV，UV只统计1次
2. **多独立窗口访问（UV模式）**：每次启动全新无痕浏览器访问一次，每个浏览器都是完全独立的会话，互不共享Cookie/缓存，每次访问都被统计为独立UV
### 两种运行模式
1. **可见窗口模式**：正常弹出Chrome浏览器窗口，可以看到访问过程，适合调试和查看效果
2. **后台静默模式**：Chrome完全在后台无头运行，不会弹出任何窗口，完全不影响电脑其他操作，行为和可见模式完全一致
### 并行加速
- 支持1-10最大并行数设置，并行数越大访问速度越快
- 大于5并行数时自动弹出确认提示，避免占用过多系统资源
- 线程池安全控制并发，所有并行任务互不干扰
## 📋 环境要求
- 全平台支持：Windows / Mac / Linux
- 已安装 Chrome 浏览器
- Python 3.6+ 环境
## 🚀 快速使用
### 一键启动（推荐）
#### Windows系统
1. 直接双击项目目录下的 `start.bat` 文件，自动启动服务
2. 等待3秒，系统会自动打开默认浏览器进入操作页面：`http://127.0.0.1:5000`
#### Mac系统
1. 直接双击项目目录下的 `start.command` 文件，自动启动服务
2. 等待3秒，系统会自动打开浏览器进入操作页面：`http://127.0.0.1:5000`
#### 通用操作步骤
1. 输入目标网址（如 `https://www.example.com`）
2. 选择功能模式（PV模式/UV模式）
3. 选择运行模式（可见窗口/后台静默）
4. 设置访问次数和最大并行数
5. 点击「开始执行任务」即可
6. 任务运行过程中可随时点击「停止任务」终止
### 停止服务
- 方法1：直接关闭运行脚本的终端/CMD窗口，服务会自动停止，所有浏览器也会自动关闭
- 方法2：在终端/CMD窗口按 `Ctrl + C` 快捷键停止

## ⚠️ 注意事项
- 工具仅供合法合规的自有网站功能测试、性能测试使用，禁止用于非法用途
- UV模式下开启高并行数时建议使用后台静默模式，避免弹出过多窗口影响操作
- 每次刷新默认随机间隔1-3秒，模拟真实用户访问行为，避免给目标网站造成压力
