# 网页自动刷新工具

一个简单易用的网页批量访问刷新工具，通过真实Chrome浏览器模拟用户操作。

## ✨ 功能特性
- 🖥️ 友好的Web操作界面
- 🌐 自定义目标网址
- 🔢 可设置刷新访问次数
- ⏸️ 支持中途停止任务
- 📊 实时进度显示
- 🚀 一键启动，无需手动敲命令

## 📋 环境要求
- Mac 操作系统
- 已安装 Chrome 浏览器
- Python 3.6+ 环境

## 🚀 快速使用
### 方法一：一键启动（推荐）
1. 直接双击项目目录下的 `start.command` 文件
2. 等待3秒，系统会自动打开浏览器进入操作页面
3. 输入目标网址（需要带`http://`或`https://`开头，例如：`https://www.example.com`）
4. 输入需要刷新的次数
5. 点击「开始执行任务」即可
6. 中途需要停止点击「停止任务」按钮即可

### 方法二：手动启动
```bash
# 进入项目目录
cd /path/to/brush_tool
# 安装依赖
pip install -r requirements.txt
# 启动服务
python app.py
# 浏览器访问 http://127.0.0.1:5000
```

## 📖 功能说明
1. **目标网址**：填写需要访问刷新的网站地址，必须带http/https前缀
2. **刷量次数**：设置总共需要刷新访问的次数
3. **开始执行**：自动启动Chrome浏览器访问目标网址，然后按设置次数自动刷新，每次刷新间隔1-3秒，模拟真实用户操作
4. **停止任务**：任务运行过程中随时可以点击停止，会自动关闭浏览器并结束任务
5. **进度显示**：实时展示当前完成次数、总次数和进度百分比

## 🛑 停止服务
- 方法1：直接关闭运行脚本的终端窗口，服务会自动停止
- 方法2：在终端窗口按 `Ctrl + C` 快捷键停止

## ❓ 常见问题
### Q1：双击start.command提示无法打开
A：这是Mac的安全限制，右键点击`start.command` → 选择「打开」即可，或者去「系统设置→隐私与安全性」，拉到底部点击「仍要打开」。

### Q2：页面打开后空白
A：按 `Cmd + Shift + R` 强制刷新页面即可，或者用无痕模式打开。

### Q3：浏览器控制台有`zijieapi.com`相关报错
A：这是你浏览器安装的字节系插件（抖音/飞书/剪映等）导致的，和本工具无关，完全不影响使用，嫌烦可以用无痕模式打开页面。

### Q4：Chrome浏览器打开后黑屏
A：已经添加了黑屏修复参数，如果还出现这个问题请升级你的Chrome浏览器到最新版本。

### Q5：启动提示端口5000被占用
A：一键启动脚本已经自动处理了残留进程，如果还是出现这个问题可以手动执行命令杀掉占用进程：`lsof -ti:5000 | xargs kill -9`

## ⚠️ 注意事项
1. 任务运行过程中不要手动关闭自动打开的Chrome窗口，否则任务会中断
2. 每次刷新间隔1-3秒，属于正常用户访问频率，不会对目标网站造成压力
3. 工具完全本地运行，不会上传任何数据，不会访问任何第三方服务器
4. 仅供合法合规的测试场景使用，请勿用于非法用途
